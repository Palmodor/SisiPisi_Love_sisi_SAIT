# Инструкция по тестированию входа и регистрации

## 🚀 Шаг 1: Запуск приложения

```bash
cd /home/proxor/Рабочий\ стол/Love_Susi/LoveSushiPMR
dotnet run
```

Приложение должно запуститься на http://localhost:5000 или https://localhost:7232

---

## 📝 Шаг 2: Тестирование регистрации

### 2.1 Валидация формы регистрации

1. Откройте страницу регистрации: `http://localhost:5000/Account/Register`
2. Попытайтесь отправить форму пустой - должны появиться русские ошибки:
   - "Это поле обязательно для заполнения"
3. Введите неправильный email (например "test") - должна ошибка:
   - "Пожалуйста, введите корректный адрес электронной почты"
4. Введите пароль менее 6 символов - должна ошибка:
   - "Пожалуйста, введите не менее 6 символов"
5. Введите разные пароли в поля пароля и подтверждения - должна ошибка:
   - "Пароли не совпадают"

### 2.2 Успешная регистрация

1. Заполните форму корректно:
   - **Имя:** Иван Петров
   - **Email:** ivan@test.ru
   - **Телефон:** +373777123456
   - **Пароль:** password123
   - **Подтверждение пароля:** password123

2. Нажмите кнопку "Зарегистрироваться"
3. Вы должны видеть сообщение об успешной регистрации
4. Вы должны быть автоматически вошли в аккаунт
5. На главной странице должно быть видно ваше имя в меню (профиль)

## 🔐 Шаг 3: Тестирование входа

### 3.1 Неправильные учетные данные

1. Откройте страницу входа: `http://localhost:5000/Account/Login`
2. Введите email: неправильный_email@test.ru
3. Введите пароль: wrongpassword
4. Нажмите кнопку "Войти"
5. Должна появиться ошибка: "Неверный email или пароль"

### 3.2 Правильные учетные данные

1. Введите email: ivan@test.ru (из регистрации)
2. Введите пароль: password123
3. Нажмите кнопку "Войти"
4. Вы должны быть переведены на главную страницу
5. В меню должно быть видно ваше имя "Иван Петров"

---

## ☑️ Шаг 4: Тестирование "Запомнить меня"

1. На странице входа установите флажок "Запомнить меня"
2. Введите правильные учетные данные
3. Нажмите "Войти"
4. Закройте браузер полностью
5. Откройте браузер и перейдите на http://localhost:5000
6. Вы должны остаться вошедшим в аккаунт (видно ваше имя в меню)
7. Профиль должен быть доступен через меню

---

## 👤 Шаг 5: Тестирование профиля

1. Будучи вошедшим, нажмите на ваше имя в меню
2. Выберите "Профиль"
3. Должна открыться страница с вашими данными:
   - Имя
   - Email
   - Телефон
   - Дата регистрации
   - Бонусный счет

---

## 🚪 Шаг 6: Тестирование выхода

1. На странице в меню нажмите на ваше имя
2. Нажмите "Выйти"
3. Вы должны быть переведены на главную страницу
4. На главной странице должна быть видна кнопка "Войти" вместо вашего имени

---

## 🛒 Шаг 7: Интеграция с корзиной

1. Будучи вошедшим, добавьте товар в корзину
2. Перейдите в корзину
3. Оформите заказ
4. Должны быть запросены адрес доставки и метод оплаты
5. Заказ должен быть сохранен в БД с userId

## 🐛 Для отладки

Если что-то не работает, откройте браузерную консоль (F12) и проверьте:

1. **Network табулка:** Проверьте запрос POST к `/Account/Login` или `/Account/Register`
   - Status должно быть 200 или 302 (редирект)
   - Response должен содержать HTML страницы

2. **Console табулка:** Проверьте наличие ошибок JavaScript
   - Если jQuery Validation не загружена, будет ошибка "$ is not defined"

3. **Application табулка → Cookies:** Проверьте наличие cookie с аутентификацией
   - Должна быть cookie с именем `.AspNetCore.Cookies`
   - Её наличие = пользователь вошел

4. **Серверные логи:** В выводе `dotnet run` должны быть логи запросов

## 📞 Контакты для поддержки

Если возникают проблемы:
1. Проверьте, что PostgreSQL запущена и доступна
2. Проверьте строку подключения в `appsettings.json`
3. Убедитесь, что миграции БД выполнены (`dotnet ef database update`)
4. Очистите кеш браузера (Ctrl+Shift+Delete)
5. Перезагрузите приложение
